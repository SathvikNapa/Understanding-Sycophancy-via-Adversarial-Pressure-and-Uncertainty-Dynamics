When $H$ > 0, the model flips 46.6\% of the time, as opposed to 23.6\% of the time when $H$ = 0, showing that flip rate is significantly affected by model confidence. \autoref{tab:rq1} shows flip rates for each model, demonstrating that this effect is consistent across all 5 models.
